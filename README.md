# MachineLearning-Web-Application



## 1. Goal of this project

 ### 당뇨 위험도 평가 서비스 개발 
 
 질병관리청에서 수행한 국민 건강 영양조사 데이터를 살펴보면, "당뇨 위험군"에 해당하는 사람이 당뇨병 환자 수 보다 2배정도 많은 것을 볼수 있습니다. 당뇨 위험군에 해당하는 사람들에게 "잠재적 당뇨 위험도"를 알리고 건강과 관련한 서비스나 제품에 대한 유입을 증가 시키기 위하여 본 서비스를 개발하였습니다.
